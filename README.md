# CodeWave Company Website

This project is a modern and responsive company website built using HTML and CSS.  
It represents a fictional web development agency called **CodeWave** and showcases services, portfolio projects, company information and contact section.

The goal of this project is to practice building real-world layouts and create a portfolio-ready website.

## Features
- Fully responsive layout
- Modern UI design
- Services, Portfolio, About and Contact sections
- Clean and organized code structure
- Built using pure HTML & CSS (no frameworks)
